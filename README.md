# Gesture Racing Game (SPYDARR RACING)

A racing game controlled using hand gestures via webcam.

## Features
- Steering using hand direction
- Speed control using hand distance
- Real-time gesture tracking
- Smooth and immersive UX

## Tech Stack
- MediaPipe (Hand Tracking)
- OpenCV
- Unity (Game Engine)

## Structure
- /docs → design & UX
- /src → code prototypes
- /assets → placeholder assets
